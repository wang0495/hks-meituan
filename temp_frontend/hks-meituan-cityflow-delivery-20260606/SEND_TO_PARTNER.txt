这是 CityFlow 珠海路线助手新版前端交付包。

建议先打开：
docs/index.html

里面有界面截图、启动方式、高德地图 Key 填写方式、真实后端联调说明。

快速预览前端：
1. 进入 frontend 文件夹
2. 运行：python -m http.server 5174 --bind 127.0.0.1
3. 打开：http://127.0.0.1:5174/

联调真实 V2 后端：
1. 确保 hks-meituan-master 与本交付包在相邻目录，或设置 CITYFLOW_BACKEND_REPO 指向 hks-meituan-master
2. 进入 frontend 文件夹
3. 运行：python -m uvicorn dev_backend:app --host 127.0.0.1 --port 8000
4. 前端顶部显示“后端在线”即连接成功

高德地图：
打开页面后，在中间地图区填写高德 Web JS API Key 和 securityJsCode，点击“加载地图”。
Key 不写在文件里，只保存在当前浏览器 localStorage。
