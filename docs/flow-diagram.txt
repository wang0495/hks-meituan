━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ ① 用户身份 + 上下文采集 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    │

━━━━━━━━━━ ⑤ Phase 1: 意图解析（上下文增强版） ━━━━━━━━━━

  SSE: event: phase | data: {"phase": "parsing"}

          │
          ▼
  ┌──────────────────────────────────────────────────────────────┐
  │  parse_intent(user_input)                                    │
  │    ├── LLM调用 (5s超时) → 结构化JSON                         │
  │    └── 规则降级 → _rule_based_parse()                        │
  │        新增: detect_emotion_need()                           │
  │          "烦"→"放松", "无聊"→"新鲜感", "累"→"恢复"          │
  │                                                              │
  │  merge_user_preference(intent, user_stated, ltm, context)    │
  │    三源融合（权重递减）:                                      │
  │    ├── 对话明确项   → 最高优先级                             │
  │    ├── LTM上下文预测 → 次优先级（仅当用户未明确表达时）      │
  │    └── 画像默认值   → 兜底                                   │
  │                                                              │
  │  输出: {                                                     │
  │    ...,                                                       │
  │    "emotion_need": "放松",                                   │
  │    "preferences_source": {         ← 每个维度的来源         │
  │      "pace": "user_stated",                                   │
  │      "budget": "user_stated",                                 │
  │      "preferences.nature": "ltm_contextual",  ← 上下文预测   │
  │      "preferences.culture": "profile_default"                 │
  │    }                                                         │
  │  }                                                            │
  └──────────────────────────────────────────────────────────────┘

━━━━━━━━━ ⑥ Phase 2+3: POI筛选 + 路线求解（动态权重版） ━━━━━━━━━

  SSE: event: phase | data: {"phase": "searching"}
  SSE: event: phase | data: {"phase": "solving"}

          │
          ▼
  ┌──────────────────────────────────────────────────────────────┐
  │  solve_route(candidates, user_intent, start_time,            │
  │              perception_ctx,                                  │
  │              dynamic_weights={       ← 来自对话+LTM+上下文   │
  │                "alpha": 1.0,          # 位移成本              │
  │                "beta": 0.3,           # 情绪收益 (对话可调)  │
  │                "gamma": 0.2,          # 疲劳惩罚 (下雨天↑)  │
  │                "delta": 0.15,         # 同类惩罚              │
  │                "budget_strictness": 1.0                       │
  │              },                                               │
  │              progress_callback=cb)    ← 驱动PipelineBar      │
  │                                                              │
  │  callback 触发 → SSE solver_stage:                           │
  │    "筛选"  → 剩余N个候选                                     │
  │    "初排"  → 路线长度N                                       │
  │    "优化"  → 第i次迭代                                       │
  │    "休息"  → 插入休息点名                                    │
  │    "收尾"  → 末站名                                          │
  └──────────────────────────────────────────────────────────────┘
