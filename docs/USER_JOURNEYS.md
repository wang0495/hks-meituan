# CityFlow 用户旅程示例

本文档展示 CityFlow 系统在不同用户场景下的完整响应流程。

---

## 旅程1: 社恐独居的周末

**用户输入**: "周末想一个人安静走走，不想去人多的地方"

**系统响应**:

### 1. 意图解析

| 维度 | 结果 |
|------|------|
| 画像匹配 | P1 社恐独居 |
| 节奏 | 闲逛型 |
| 社交倾向 | 0.1 |
| 硬约束 | 排队容忍度<5min, 避开人流高峰 |

### 2. 路线规划

| 序号 | POI | 时间 | 宁静度 | 文化深度 |
|------|-----|------|--------|----------|
| 1 | 珠海博物馆 | 09:00-11:00 | 0.8 | 0.95 |
| 2 | 情侣路海滨 | 11:30-13:00 | 0.7 | 0.6 |
| 3 | 安静咖啡馆 | 13:30-15:00 | 0.9 | 0.2 |

### 3. 文案生成

> "周末给自己一个安静的角落，让心灵慢下来。这趟行程是为你量身定制的。"

### 4. 情绪曲线

```
宁静度
  0.9 |           *
  0.8 |     *
  0.7 |           *
  0.6 | *
  0.5 |
      +--+---------+---> 时间
        开始  中间  结束
```

- 开始: 宁静 0.6
- 中间: 宁静 0.8, 文化 0.9
- 结束: 宁静 0.9
